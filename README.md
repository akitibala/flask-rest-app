# flask-rest-app

Prerequisite : 
- Add csvs unzipped file(with csv files) in root directory for working solution
- python3.8
- flask
- jsonschema

## Run

### Method 1:

`python app.py`

### Method 2 :

` docker-compose build` and ` docker-compose up -d `

***Make sure you docker and docker-compose installed***
